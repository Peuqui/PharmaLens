SSL-Zertifikat für Drug2QR installieren

Wichtig: Dieses Zertifikat ist nur für den lokalen Gebrauch im Heimnetzwerk gedacht!

Schritt 1: CA-Zertifikat herunterladen

Lade das CA-Zertifikat herunter:

CA-Zertifikat herunterladen

Schritt 2: Installation auf dem Mobilgerät

iOS (iPhone/iPad):

  1. Öffne die heruntergeladene Datei
  2. Gehe zu Einstellungen → Allgemein → VPN & Geräteverwaltung
  3. Tippe auf das Profil "drug2qr-ca"
  4. Tippe auf "Installieren" und gib dein Passcode ein
  5. Gehe zu Einstellungen → Allgemein → Info → Zertifikatsvertrauenseinstellungen
  6. Aktiviere den Schalter bei "drug2qr-ca"

Android:

  1. Öffne die heruntergeladene Datei
  2. Gib dem Zertifikat einen Namen (z.B. "Drug2QR")
  3. Wähle "VPN und Apps" oder "WLAN" als Verwendungszweck
  4. Tippe auf "OK"

Schritt 3: App öffnen

Nach der Installation kannst du die App über HTTPS aufrufen:

  • https://192.168.0.1:5173
  • https://localhost:5173
